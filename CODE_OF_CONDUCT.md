# Contributor code of conduct

We want the **GovAI Core** community to stay a respectful, technical, and productive place to collaborate.

## Our standards

- **Be professional.** Disagreement is normal; personal attacks, harassment, and hate are not.
- **Assume good intent** when tone is ambiguous, and clarify before escalating.
- **Stay on topic** in issues and pull requests so maintainers and contributors can move work forward.
- **Respect boundaries.** Do not spam, scrape contacts, or pressure individuals for immediate responses.

## Scope

This applies to all community spaces for this project (issues, pull requests, discussions, chat channels we officially link to, and related events).

## Reporting

If you see behavior that violates this code of conduct, report it to the maintainers (for example via a private GitHub report or the contact path described in `SECURITY.md` when sensitive). We will review reports in good faith and respond as we are able.

## Enforcement

Maintainers may remove, edit, or reject contributions or comments, or temporarily or permanently block accounts, when conduct undermines a safe and productive environment. Serious cases may be escalated to platform support.

This document is a concise project-specific code of conduct inspired by common open source community norms. It is not legal advice.
