# Community Guidelines

GovAI aims to maintain a technically rigorous and contributor-friendly open source community.

## Expectations

Contributors should:
- communicate respectfully
- focus on technical clarity
- provide actionable feedback
- explain reasoning clearly
- avoid hostile review behavior

## Review culture

Good reviews are:
- specific
- reproducible
- operationally grounded
- technically justified

## Governance-oriented mindset

GovAI prioritizes:
- deterministic behavior
- fail-closed enforcement
- auditability
- traceability
- operational correctness
