# GovAI Roadmap

GovAI is an evidence-first governance layer for AI systems focused on decision-level auditability, fail-closed enforcement, evidence continuity, and operational AI governance.

## Phase 1: Enforcement core

- Harden VALID / INVALID / BLOCKED decision semantics
- Improve fail-closed enforcement behavior
- Strengthen evidence completeness checks
- Improve approval and promotion gates
- Improve readiness diagnostics

## Phase 2: Evidence and auditability

- Signed evidence bundle verification
- Portable evidence replay
- Immutable audit exports
- Deterministic export manifest verification
- Audit event schema versioning

## Phase 3: Runtime governance

- Runtime policy enforcement
- Governance API hardening
- Tenant-scoped RBAC
- Runtime traceability
- Production deployment guidance

## Phase 4: Policy engine

- Policy pack loading abstraction
- Governance policy schema validation
- Governance-as-code examples
- AI Act mapping policy pack

## Phase 5: Integrations

- GitHub Actions examples
- OpenTelemetry trace propagation
- MLflow evidence export example
- LangChain evidence adapter

## Phase 6: OSS developer experience

- Contributor quickstart
- Local fail-closed demo
- Docker Compose setup
- Architecture diagram
- Issue templates
- PR template
