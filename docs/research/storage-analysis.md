# Storage analysis

Storage characteristics combine **measured** average bytes per event (from a sampled chain) with **analytical** projections for monthly ledger growth, index overhead, and checkpoint overhead. Multipliers (for example index overhead) are documented assumptions, not measured from a live Postgres instance.

See `storage-benchmarks.json` for the latest computed values.
