# Evidence pack examples (pointers)

Evidence packs are the **machine-readable** artefacts used for artefact-bound CI (`<run_id>.json` + **`evidence_digest_manifest.json`**).

## Canonical documentation

- **[docs/evidence-pack.md](../../docs/evidence-pack.md)** — `govai evidence-pack init`, submit/verify flow
- **[docs/golden-path.md](../../docs/golden-path.md)** — deterministic **`BLOCKED` → `VALID`** narrative

## Non-production reminder

Example **`run_id`** strings in docs are placeholders. Do not paste production keys or customer data into example repos.
