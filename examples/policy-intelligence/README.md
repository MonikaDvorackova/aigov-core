# Policy intelligence examples

Stdlib-only drivers for Phase 18 tooling. Run from anywhere; scripts `cd` to the repository root.

## Sample snapshot

- [`sample-governance-control-snapshot.json`](sample-governance-control-snapshot.json) — valid example input for scoring and Markdown report generation.

## Shell drivers

| Script | Purpose |
|--------|---------|
| [`run-policy-intelligence-check.sh`](run-policy-intelligence-check.sh) | Emit diagnostics JSON |
| [`run-policy-coverage-score.sh`](run-policy-coverage-score.sh) | Emit coverage and maturity scores |
| [`run-governance-control-report.sh`](run-governance-control-report.sh) | Print Markdown report to stdout |
