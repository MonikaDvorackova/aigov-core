# Security examples (threat matrix)

The file `sample-threat-matrix.json` lists adversary capabilities, mitigations, residual risks, and in-scope / out-of-scope notes for eight governance-relevant threats.

Validate with:

```bash
python3 scripts/threat_model_check.py --json
make threat-model-check
```

See `docs/security/threat-matrix.md` and `docs/security/threat-model-summary.md`.
